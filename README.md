# REBUS Messenger

Responsive web messenger for secure communication inside REBUS contours.

## Current stage

Initial responsive login/splash page:

- desktop background variants: 1920, 2560, 3840;
- mobile background variants: 1080, 1440;
- adaptive layout for desktop, tablet, and phones;
- REBUS cyberpunk visual language.
